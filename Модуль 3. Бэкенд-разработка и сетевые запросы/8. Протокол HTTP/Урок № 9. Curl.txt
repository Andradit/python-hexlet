Используя curl выполните запрос на получение списка пользователей к localhost на порт 8080 по пути /users. В заголовке
запроса сообщите серверу, что ожидаете ответа в формате JSON (тип application/json)

Если запрос успешный, то в терминале упражнения вы увидите список пользователей, который вернул сервер

Получившуюся команду запишите в файл solution

Подсказки
Заголовок Accept, который позволяет указать, какой тип данных мы ожидаем получить в ответе

curl -H "Accept: application/json" localhost:8080/users

SOLUTION

curl localhost:8080/users -H "Accept: application/json"
