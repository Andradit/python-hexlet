Форма состоит из пяти основных элементов: - два однострочных текстовых поля для ввода имени и email - большое текстовое поле (textarea) для ввода сообщения - чекбокс с принятием условий сервиса - кнопка отправки сообщения index.html - заголовок первого уровня внутри формы: «Обратная связь» - каждое текстовое поле формы лежит в обёртке с классом .form-input.
